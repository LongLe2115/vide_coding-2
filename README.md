## Library Management UI (HTML/CSS/JS)

Chạy demo giao diện quản lý thư viện theo đề bài, dùng **HTML/CSS/JS thuần** và lưu dữ liệu tạm bằng **localStorage**.

### Cách chạy
- Mở file `index.html` bằng trình duyệt (Chrome/Edge).

### Tài khoản mẫu
- **Admin**: `admin` / `admin`
- **Thủ thư**: `librarian` / `librarian`

### Ghi chú
- Đây là bản demo UI + logic phía client để bạn nắm luồng nghiệp vụ (CRUD, mượn/trả, báo cáo).
- Khi làm đồ án hoàn chỉnh, bạn có thể thay `localStorage` bằng API + CSDL.
